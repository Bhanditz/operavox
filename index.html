Operavox

Operavox is a accessibility extension for Opera Android products which makes use of Android's Talkback framework to provide accessibility support.

Getting the Source

The current development snapshots and source is available at Opera Software's Chromevox repository and Operavox repository.

Versions

The versions listed here are only published for reference and testing purposes. While the actual published versions are identical to the releases provided here, official releases will use our official distribution channel instead of the Github static page.

  • 1.0 for Opera Devices SDK {{TBD}} and Opera Mobile {{TBD}} for Android

Testing

To test against a specific version, you will need to copy the URL of a version listed above and initialize with the flag accessibility-js-url set to the URL of the version that you are testing against. A mismatching version may work or result in completely undesired behavior. Please note that our internal testing is only done against matching versions.

A example would be: 

--accessiblity-js-url=https://operasoftware.github.io/operavox/1.0/operavox.js

Depending on the version, the extension may or may not work on Chrome for Android.
